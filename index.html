ABOUT

100% workshop technology conference taking place at the end of summer, August 28-29th, 2015 at the University of Social Sciences and Humanities in Warsaw, Poland. If you believe in learning by doing approach and are thirsty for knowledge and networking this event is for you!

There will be multiple hands-on sessions on software development, devops tools, product design, IoT and hardware renaissance, agile practices and many, many more. Instead of just talking about great new tools - Codepot will show you how to actually use them.

React-native: building native iOS apps with javascript

At this workshop you will implement a basic native (kind of) iOS app using Facebook's React Native framework.

What is React Native:

React-Native is Facebook's React based native application framework. You can write iOS applications using native Views and Controls, while the whole application logic is written in Javascript (the good parts of it in fact). Soon Android version will be released but for now we can build iOS apps with it.

It's true embodiment of Attwood's Law: any application that can be written in JavaScript, will eventually be written in JavaScript.

Who is the ideal participant:

  • mobile developers (iOS/Android) who wants to learn to develop iOS (later Android) apps differently
  • Javascript developer who wants to write native iOS apps (Android in the future) without learning Obj-C
